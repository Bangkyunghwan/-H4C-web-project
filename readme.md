> 이 레포지토리는 **H4C(P4C) 웹 해킹 트랙 스터디**의 최종 프로젝트로 진행한 **보안 게시판(Secure Board)**입니다.
>
> 스터디에서 학습한 웹 해킹 공격 기법(SQL Injection, XSS, CSRF 등)을 방어하는 시큐어 코딩(Secure Coding)을 적용하여 **PHP**로 직접 게시판을 구현했습니다.

<br><br>

## 📌 프로젝트 개요

*  **주제:** H4C 웹 해킹 트랙 최종 프로젝트 (시큐어 코딩 적용 게시판)
*  **개발 언어:** PHP
*  **주요 기능:**
    *  회원가입 및 로그인 (비밀번호 해싱 적용)
    *  게시글 작성, 조회, 수정, 삭제 (CRUD)
    *  XSS, SQL Injection 등 주요 웹 취약점 방어 로직 적용
